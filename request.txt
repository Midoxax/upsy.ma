# U.Psy — Feature Request Workflow

## How to Use This File

1. Write a detailed specification for any new feature, refactor, or fix below.
2. Reference the `project_map.md` for architecture context.
3. Ask the AI to execute the specification.
4. After implementation, move the request to the "Completed" section with a date.
